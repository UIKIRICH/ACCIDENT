# 事故处理平台前端 Demo

这是一个纯前端、可交互的动态网站示例，参考你提供的网页排版进行了重制和美化。

## 特性
- 单文件入口 SPA（`index.html`）
- 覆盖页面：
  - 登录 / 注册
  - 首页总览
  - 事故录入
  - 视频处理
  - 智能分析
  - 责任建议
  - 规则依据
  - 人工复核
  - 历史案例
  - 任务中心
  - 规则库管理
- 支持：
  - 侧边导航切换
  - 深浅色主题切换
  - 表单录入与草稿保存
  - 拖拽/点击上传文件
  - 关键帧切换
  - 历史案例筛选
  - 人工复核归档
  - Toast 消息提示
- 已预留后端接口替换位置：`api.js`

## 使用方法
直接双击 `index.html`，或放到任意静态服务器运行。

## 后端对接说明
当前数据由 `mock-data.js` + `api.js` 模拟。
后续只需将 `api.js` 中的 mock 方法替换为真实 `fetch('/api/...')` 即可。
